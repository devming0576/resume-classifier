# 贡献指南

感谢你对 Resume Classifier 项目的关注与贡献！

## 如何贡献
1. Fork 本仓库
2. 创建功能分支
3. 完成你的修改
4. 添加/更新测试用例
5. 运行测试和 pre-commit 钩子
6. 如有需要，更新文档
7. 提交 Pull Request（PR）

## 代码规范
- 遵循 PEP 8
- 使用类型注解（type hints）
- 为所有公开函数/类编写 docstring
- 提交前运行 `black`、`flake8`、`isort`

## PR 流程
- 确保所有测试通过
- 清晰描述你的更改内容
- 如有相关 issue，请在 PR 中引用
- 等待代码审核并根据反馈修改

## 测试
- 为新功能添加单元/集成测试
- PR 前请运行 `pytest`

## 文档
- 在 `docs/` 目录下更新或添加相关文档
- 新功能请补充用法示例
