## 简介
本项目提供自动化脚本，简化常见开发任务，提高效率。

---

## clean.sh 脚本
- 作用：清理构建产物、缓存、临时文件。
- 用法：
  ```bash
  ./clean.sh
  ```
- 建议在每次大改动或发布前运行。

---

## test.sh 脚本
- 作用：一键运行所有单元测试和代码检查。
- 用法：
  ```bash
  ./test.sh
  ```
- 可用于本地模拟 CI 流程。

---

## 最佳实践
- 脚本需赋予可执行权限：`chmod +x clean.sh test.sh`
- 可在 CI/CD 流程或 pre-commit 钩子中集成。
- 如需自定义，建议保留原始功能。
